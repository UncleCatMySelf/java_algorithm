## Java_Algorithm（Java算法集合）

### 学习算法是为了什么？
1、应对大型IT公司的算法面试题；
2、IDE即对编译算法的封装；
3、搜索引擎中对几千、几亿数据进行优劣排序；
4、游戏对算法的引用是非常丰富的；
5、算法对性能优化的推进；
6、白板编程面试主要考察算法基本思想的基本功（一般是十行内的代码）

### 项目目录

1、线性（排序）
2、树形结构
3、图形结构
